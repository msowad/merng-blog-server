## Blog app made with Nodejs, React, Graphql and MongoDB

### Run this project

In the project directory, you can run:

```shell
npm install
```

```shell
cp .env.example .env
```

```shell
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
